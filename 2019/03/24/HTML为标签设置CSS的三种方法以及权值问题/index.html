外部式(外联式)

外联式是把css代码写到一个单独的外部文件中，然后文件名是以“.css”为扩展名，在<head>标签内用<link>标签将css样式代码文件链接到HTML文件内。

注意：
rel=”stylesheet” type=”text/css”是固定写法不能修改

三种方法的优先级

优先级：内联式>嵌入式>外部式
注意：嵌入式>外部式有一个前提：嵌入式css样式的位置一定在外部式的后面
其实就是就近原则（离被设置元素越近优先级越高）
但是上面总结的优先级是在内联式、嵌入式、外部式样式表中css样式是在相同权值的情况下的

css样式的权值

浏览器是根据权值来判断使用哪种CSS样式的，哪种权值高就使用哪种
规则：标签的权值为1，类选择符的权值为10，ID选择的权值最高为100
